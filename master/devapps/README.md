# master/devapps

## 📝 Description

Ce module gère les applications personnalisées développées pour les besoins spécifiques du projet.



## 📁 Structure

- `models.py` :  Définit les modèles pour les applications personnalisées.
- `admin.py` :  Configure l'interface d'administration pour ces applications.
- `forms.py` : Contient les formulaires pour la création et la modification des applications.
- `views.py` : Gère les vues pour l'affichage et la gestion des applications.
- `templates/devapps/` : Contient les templates HTML pour les vues.

## 🔐 Fonctionnalités

- Création et gestion des applications personnalisées.
- Gestion des permissions et des groupes
- Intégration avec le tableau de bord utilisateur
